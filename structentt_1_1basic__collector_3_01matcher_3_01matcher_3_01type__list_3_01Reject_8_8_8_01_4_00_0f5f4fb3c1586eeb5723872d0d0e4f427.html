Detailed Description

template<typename... Reject, typename... Require, typename... Rule, typename... Other>
struct entt::basic_collector< matcher< matcher< type_list< Reject... >, type_list< Require... > >, Rule... >, Other... >

Collector.

A collector contains a set of rules (literally, matchers) to use to track entities.
Its main purpose is to generate a descriptor that allows an observer to know how to connect to a registry.

Template Parameters
AnyOfTypes of components for which changes should be detected.
MatcherTypes of grouping matchers.
