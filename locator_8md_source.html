14 # Introduction
15 
16 Usually service locators are tightly bound to the services they expose and it's
17 hard to define a general purpose solution. This template based implementation
18 tries to fill the gap and to get rid of the burden of defining a different
19 specific locator for each application.<br/>
20 This class is tiny, partially unsafe and thus risky to use. Moreover it doesn't
21 fit probably most of the scenarios in which a service locator is required. Look
22 at it as a small tool that can sometimes be useful if users know how to handle
23 it.
24 
25 # Service locator
26 
27 The API is straightforward. The basic idea is that services are implemented by
28 means of interfaces and rely on polymorphism.<br/>
29 The locator is instantiated with the base type of the service if any and a
30 concrete implementation is provided along with all the parameters required to
31 initialize it. As an example:
32 
33 ```cpp
34 // the service has no base type, a locator is used to treat it as a kind of singleton
35 entt::service_locator<my_service>::set(params...);
36 
37 // sets up an opaque service
38 entt::service_locator<audio_interface>::set<audio_implementation>(params...);
39 
40 // resets (destroys) the service
41 entt::service_locator<audio_interface>::reset();
42 ```
43 
44 The locator can also be queried to know if an active service is currently set
45 and to retrieve it if necessary (either as a pointer or as a reference):
46 
47 ```cpp
48 // no service currently set
49 auto empty = entt::service_locator<audio_interface>::empty();
50 
51 // gets a (possibly empty) shared pointer to the service ...
52 std::shared_ptr<audio_interface> ptr = entt::service_locator<audio_interface>::get();
53 
54 // ... or a reference, but it's undefined behaviour if the service isn't set yet
55 audio_interface &ref = entt::service_locator<audio_interface>::ref();
56 ```
57 
58 A common use is to wrap the different locators in a container class, creating
59 aliases for the various services:
60 
61 ```cpp
62 struct locator {
63  using camera = entt::service_locator<camera_interface>;
64  using audio = entt::service_locator<audio_interface>;
65  // ...
66 };
67 
68 // ...
69 
70 void init() {
71  locator::camera::set<camera_null>();
72  locator::audio::set<audio_implementation>(params...);
73  // ...
74 }
75 ```
