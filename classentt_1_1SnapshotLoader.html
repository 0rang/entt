Detailed Description

template<typename Entity>
class entt::SnapshotLoader< Entity >

Utility class to restore a snapshot as a whole.

A snapshot loader requires that the destination registry be empty and loads all the data at once while keeping intact the identifiers that the entities originally had.
An example of use is the implementation of a save/restore utility.

Template Parameters
EntityA valid entity type (see entt_traits for more details).

template<typename Entity >
const SnapshotLoader& entt::SnapshotLoader< Entity >::orphans ( ) const
inline

Destroys those entities that have neither components nor tags.

In case all the entities were serialized but only part of the components and tags was saved, it could happen that some of the entities have neither components nor tags once restored.
This functions helps to identify and destroy those entities.
